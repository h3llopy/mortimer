Features

This module hide the advance search options like "Filters", "Group By" and "Favorites" For the Particular Group of Users

Configuration

Hide the advanced search like "Filters", "Group By" and "Favorites" button for a particular group of users so for that, you have to go to Settings --> Users & Companies --> Users then open the user and mark the "Hide Advanced Search Button"(You can see this configuration after activate the debug mode).
